Instalacija i pokretanje

-preuzeti source kod
-unzipati folder igrice
-premjestiti folder "Resources" u folder "Release"
-pokrenuti "Doodlesfml" file u "Release" folderu

Problemi tijekom izrade

-prvi susret sa rukovanjem sa stackom
-upoznavanje s brojnim SFML funkcijama i načinima na koji se koriste


Moguća poboljšanja

-bolji gui-a(unapređivanje glavnog izbornika, dodavanje game-over izbornika)
-kopleksnost same igrice- dodati neprijatelje, više vrsti platformi, levele
-score- zapamtiti high score, ispisati ga na game-over izboriku
-izgled igrice-animacije
